Getting Startet

Quick start

To install and use the components for your Svelte App, type in:

npm i svelte-cybercomp

Alternatively you can download the code from github.com and include it manually. Only do it, if you have technical limitation and not because of design limitation. These components are build with desing-freedom in mind.

To read the documentation, click one of the links to the left.
